# 🎓 Student Services Support Chatbot

An AI-driven conversational assistant designed to streamline communication between students and college administration. This project automates responses to frequently asked queries, ensuring students get instant support for their academic and administrative needs.

## 🚀 Key Features
- **Smart Query Handling:** Provides instant answers to common questions regarding:
  - 💰 **Fees:** Payment deadlines and portal navigation.
  - 📚 **Library:** Operating hours and facility guidelines.
  - 📝 **Exams:** Schedule updates and academic calendar info.
  - 💼 **Internships:** TPO cell contact and application procedures.
- **Modern UI:** Built with **Streamlit** for a clean, intuitive, and responsive web interface.
- **Efficient Logic:** Utilizes optimized pattern-matching algorithms to deliver accurate results in milliseconds.

## 🛠 Tech Stack
- **Programming Language:** Python
- **Web Framework:** Streamlit
- **Logic:** Natural Language Pattern Matching

## 💻 How to Run This Project
1. **Clone the repository:**
   git clone https://github.com/tanishq-chauhan750/Student-Chatbot.git
 2. **Install dependencies:**
